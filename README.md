## Publicação (GitHub Pages)
1. Crie um repositório novo (ex.: `bar-da-carmen`).
2. Faça upload de todos os arquivos desta pasta (ou extraia o zip e arraste).
3. Em **Settings → Pages**, selecione `Branch: main` e `Folder: /(root)`.
4. Abra a URL do Pages no Android (Chrome) e instale como app.
